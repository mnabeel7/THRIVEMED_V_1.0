# ThriveMed - RCM & Medical Billing Services

ThriveMed is a professional web application providing Revenue Cycle Management (RCM) and comprehensive Medical Billing Services for healthcare providers. This platform highlights key services including Accounts Receivable (AR) Recovery, Physician Billing, Laboratory Billing, and more.

## Features

- **Modern UI/UX**: Built with React and designed for a professional aesthetic.
- **Service Pages**: Dedicated pages detailing RCM, AR Recovery, Physician Billing, and Laboratory Billing.
- **Responsive Design**: Optimized for all devices and screen sizes.
- **Interactive Elements**: Includes smooth scrolling, animations, and toast notifications.

## Technologies Used

- React.js (v18)
- React Router DOM
- Swiper (Carousels)
- AOS (Animations)
- EmailJS (Contact Forms)
- Tailwind / CSS Modules (Styling)

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <your-github-repo-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd THRIVEMED
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the App Locally

To start the development server, run:
```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

To create an optimized production build, run:
```bash
npm run build
```

This will output the static files in the `build` directory, ready to be deployed on platforms like Vercel, Netlify, or GitHub Pages.

## Deployment

This project is configured to be seamlessly deployed on [Vercel](https://vercel.com). Just connect your GitHub repository to Vercel, and it will automatically handle the build process using `npm run build`.
